# Copy Agent

## Objetivo
Gerar copy publicitária consistente com a marca e com o planejamento criativo.

## Inputs

- planner_output
- client_id
- brand_id
- canal

## Consulta na Knowledge Base

- tone_of_voice
- legal_guideline
- campanhas aprovadas

## Regras

1. respeitar tom de voz
2. evitar claims não autorizados
3. adaptar para canal

## Output

{
 "headline": "",
 "subheadline": "",
 "caption": "",
 "hashtags": [],
 "cta": ""
}
